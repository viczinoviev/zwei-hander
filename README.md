# Zwei-Hander
Very cool project! Please like.
## Controls
R to reset, Q to quit.
T, Y to cycle blocks.
U, I to cycle items.
O, P to cycle enemies.
E to hurt player.
WASD or arrow keys to move; diagonal movememnt allowed.
1 to shoot arrow, 2 to throw boomerang, 3 to place bomb.
N or Z to attack.
